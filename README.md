# University Compass 🧭

University Compass는 대학생들의 학업 집중력 향상과 효율적인 정보 관리를 돕기 위해 기획된 모바일-웹 MVP(Minimum Viable Product)입니다. 
넘쳐나는 정보 속에서 불필요한 알림을 줄이고, 나에게 꼭 필요한 학사 정보와 진로 탐색 기능을 제공하며, '방해 금지' 타이머를 통해 몰입 환경을 만들어줍니다.

## 🌟 주요 기능

1. **대시보드 (Home)**
   - 오늘 하루 집중한 총 시간 확인
   - 임박한 학사/개인 일정 디데이(D-Day) 제공
   - 오늘의 핵심 목표(Top 3) 설정 및 달성 체크

2. **집중 타이머 (Focus Session)**
   - 과목별 타이머 설정 기능 (뽀모도로 스타일, 기본 25분)
   - 타이머 작동 시 하단 네비게이션이 숨겨지고 다크 모드(Focus Mode)로 전환되어 시각적 방해요소 차단

3. **핵심 학교 정보 (Info)**
   - 장학금, 취업/인턴, 수업/학사 등 필수 정보만 큐레이션하여 제공
   - 긴급/중요도에 따른 태그 분류로 빠른 정보 파악 가능

4. **진로 나침반 (Career)**
   - 나만의 성향을 파악할 수 있는 커리어 패스 진단
   - 프론트엔드, 백엔드, 데이터 분석, 기획 등 관심 직무 탐색

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Styling**: Tailwind CSS (CDN 연동)
- **Icons**: FontAwesome (CDN 연동)
- **Data Visualization**: Chart.js (CDN 연동, 확장을 위해 포함)

## 📁 프로젝트 구조

단일 페이지 애플리케이션(SPA) 형태로 구현되어 있으며, JavaScript를 통해 각 화면(Screen)의 활성화/비활성화(`display: none` / `block`)를 제어합니다.

- `index.html`: UI 구조, 스타일링(CSS), 비즈니스 로직(JS)이 포함된 메인 파일
- `.gitignore`: Git 버전 관리에서 제외할 항목 명시

## 🚀 실행 방법

별도의 빌드 과정 없이, 프로젝트 폴더 내의 `index.html` 파일을 브라우저로 열면 즉시 실행해 볼 수 있습니다.
모바일 디바이스(가로 폭 430px 이하) 환경에 최적화되어 있습니다. 브라우저의 개발자 도구(F12)에서 모바일 뷰로 확인하는 것을 권장합니다.
